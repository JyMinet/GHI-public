v0.4.2 — Sandbox Engine + API v1.1

La version v0.4.2 du Sandbox Engine fournit un modèle synthétique plus cohérent, unifié et aligné avec la nouvelle API publique v1.1. La mise à jour est complètement rétrocompatible.

Principales évolutions

  • API v1.1 (stable) : structure unifiée, métadonnées enrichies.
  • Nouveaux endpoints : /metadata, /network, /forecast.
  • Données synthétiques améliorées : coûts régionaux harmonisés, historique revu.
  • Compatibilité CI totale : modèles stabilisés, cohérence renforcée.
  • Dépréciation interne : suppression du routeur API v1.0.
  • Docs synchronisées : ENGINE_DESIGN_CHANGES_v0.4.2.md, API_v1.1_Spec.md.

Statut

  • Sandbox Engine : v0.4.2 actif
  • API publique : v1.1 stable
  • API v1.0 : legacy (maintenue mais figée)

v0.4.2 — Sandbox Engine + Public API v1.1

Version v0.4.2 of the Sandbox Engine delivers a more coherent synthetic model and ships with the stable public API v1.1. Fully backward-compatible.

Main changes

  • API v1.1: unified format, enriched metadata, normalized region definitions.
  • New endpoints: /metadata, /network, /forecast.
  • Improved synthetic model: harmonized costs, reworked /history.
  • CI-compliant deterministic output.
  • Deprecated: internal v1.0 router removed.
  • Docs updated: ENGINE_DESIGN_CHANGES_v0.4.2.md, API_v1.1_Spec.md.

Status

  • Sandbox Engine: v0.4.2 active
  • Public API: v1.1 stable
  • API v1.0: legacy
